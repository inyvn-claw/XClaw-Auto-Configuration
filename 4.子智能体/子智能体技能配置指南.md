# XClaw 科研子智能体技能配置指南

> 基于 [awesome-openclaw-skills](https://github.com/VoltAgent/awesome-openclaw-skills) 和 [LabClaw](https://github.com/wu-yc/LabClaw) 优化配置
> 覆盖完整论文工作流：选题 → 调研 → 设计 → 实验 → 写作 → 审稿 → 修改

---

## 📋 子智能体与技能分配总览

| 子智能体 | 英文标识 | 核心职责 | 技能数量 | 论文工作流阶段 |
|---------|---------|---------|---------|--------------|
| **文献调研员** | researcher | 文献检索、知识提取、趋势分析 | 12 | 选题、调研 |
| **创意生成器** | idea | 创新点挖掘、研究方向建议 | 8 | 选题、开题 |
| **导师审核员** | mentor | 方案评估、风险识别、学术指导 | 7 | 开题、审稿、修改 |
| **架构设计师** | architect | 系统设计、技术选型、模块划分 | 10 | 设计、实现 |
| **实验工程师** | coder | 代码实现、实验搭建、调试优化 | 14 | 实现、实验 |
| **论文撰写员** | writer | 学术写作、图表生成、格式规范 | 16 | 写作、排版 |
| **论文审稿人** | reviewer | 质量检查、逻辑审查、改进建议 | 8 | 审稿、修改 |
| **科研主管** | coordinator | 任务分配、进度跟踪、整体协调 | 7 | 全流程管理 |

## 1. 文献调研员 (researcher)

**职责定位**：科研信息的收集者和整理者，为选题和写作提供知识基础

**论文工作流阶段**：选题阶段 → 文献综述撰写

### 核心技能

#### 🔴 必装技能

| 技能名称 | 来源 | 功能描述 | 安装命令 |
|---------|------|---------|---------|
| **arxiv-watcher** | ClawHub | ArXiv 论文搜索和摘要总结 | `clawhub install arxiv-watcher` |
| **literature-review** | ClawHub | 文献综述撰写辅助 | `clawhub install literature-review` |
| **deep-research** | ClawHub | 深度研究代理，多步骤复杂调研 | `clawhub install deep-research` |
| **pubmed-search** | LabClaw | PubMed 生物医学文献检索 | 已随 LabClaw 安装 |
| **scientific-writing** | LabClaw | 科学写作规范和指导 | 已随 LabClaw 安装 |

#### 🟡 推荐技能

| 技能名称 | 来源 | 功能描述 | 安装命令 |
|---------|------|---------|---------|
| **aclawdemy** | ClawHub | AI 代理的学术研究平台 | `clawhub install aclawdemy` |
| **exa-web-search-free** | ClawHub | 高质量学术内容神经网络搜索 | `clawhub install exa-web-search-free` |
| **notebooklm-skill** | ClawHub | Google NotebookLM 笔记查询 | `clawhub install notebooklm-skill` |
| **citation-management** | LabClaw | 引文管理和格式化 | 已随 LabClaw 安装 |
| **google-scholar-search** | LabClaw | Google Scholar 学术搜索 | 已随 LabClaw 安装 |
| **semantic-scholar** | LabClaw | Semantic Scholar 语义搜索 | 已随 LabClaw 安装 |

### 典型工作流

```
1. 使用 arxiv-watcher 检索最新论文
   → "搜索 ArXiv 上关于 'transformer attention mechanism' 的最新 20 篇论文"

2. 使用 deep-research 进行系统性调研
   → "深入研究对比学习在计算机视觉中的应用现状"

3. 使用 literature-review 生成文献综述
   → "帮我撰写关于扩散模型的文献综述"

4. 使用 citation-management 管理参考文献
   → "将检索到的论文按主题分类并生成引用列表"
```

## 📊 完整论文工作流技能调用链

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           论文工作流技能调用链                                │
└─────────────────────────────────────────────────────────────────────────────┘

阶段 1: 选题与开题
─────────────────
Coordinator   → 启动 researcher 进行文献调研
                 ├─ researcher: arxiv-watcher, literature-review, deep-research
                 └─ 输出: 文献综述

Coordinator   → 启动 idea 生成创新点
                 ├─ idea: agent-news, trend-analysis, brainstorming
                 └─ 输出: 创新方向列表

Coordinator   → 启动 mentor 审核选题
                 ├─ mentor: peer-review-simulator, risk-assessment, perplexity
                 └─ 输出: 选题可行性报告


阶段 2: 系统设计与架构
```
