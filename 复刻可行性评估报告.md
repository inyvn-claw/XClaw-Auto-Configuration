## ⚠️ 需要改进的地方

### 1. 🔴 高优先级

#### 1.1 创建一键安装脚本

**现状：** 需要手动执行多个安装命令
**建议：** 创建 `install.sh` 脚本

```bash
#!/bin/bash
# install.sh - 一键安装脚本

echo "=== XClaw 配置安装脚本 ==="

# 1. 检查 OpenClaw 是否安装
if ! command -v openclaw &> /dev/null; then
    echo "❌ OpenClaw 未安装，请先安装"
    exit 1
fi

# 2. 复制子智能体配置
echo "[1/4] 复制子智能体配置..."
cp -r "4.子智能体/workspace-"* ~/.openclaw/

# 3. 安装技能
echo "[2/4] 安装技能..."
cd ~/.openclaw/skills
if [ ! -d "LabClaw" ]; then
    git clone https://github.com/wu-yc/LabClaw.git
    cp -r LabClaw/skills/* .
fi

echo "[3/4] 安装 ClawHub 技能..."
clawhub install arxiv-watcher --force 2>/dev/null || true
clawhub install literature-review --force 2>/dev/null || true
# ... 其他技能

echo "[4/4] 完成！"
echo "请配置 API Key: export MATON_API_KEY='your-key'"
```

```bash
#!/bin/bash
# validate-config.sh

echo "=== 配置验证 ==="

# 检查必要文件
echo "[1/3] 检查必要文件..."
for file in "README.md" "4.子智能体/agents_config.json"; do
    if [ -f "$file" ]; then
        echo "  ✓ $file"
    else
        echo "  ✗ $file 缺失"
        exit 1
    fi
done

# 检查 JSON 有效性
echo "[2/3] 验证 JSON 文件..."
python3 -m json.tool "4.子智能体/agents_config.json" > /dev/null && echo "  ✓ agents_config.json 有效"

# 检查敏感信息
echo "[3/3] 检查敏感信息..."
if grep -r "api_key\|token\|secret" --include="*.json" --include="*.md" . 2>/dev/null | grep -v "your_\|example\|placeholder"; then
    echo "  ⚠ 发现可能的敏感信息，请检查"
else
    echo "  ✓ 未发现明显敏感信息"
fi

echo "=== 验证完成 ==="
```
